## index.html
- Només entrar surt un formulari on l'alumne pot descarregar l'script SQL per carregar la base de dades MySQL al seu SGBD.
- Si posa el codi d'un repte i li dona a "Començar l'aventura", si el repte existeix, entrarà i començarà al nivell 1 del repte corresponent.

## admin.html:
- Per crear nous reptes, posem tota la informació necessària, creem les preguntes i ho pugem.

## assoliments.html:
- Quan l'alumne acaba un repte satisfactòriament, se li pot donar opcionalment una enganxina en la pàgina d'assoliments (si quan s'ha creat el repte, s'ha configurat aquesta part).

## monitor.html:
- Per veure els alumnes que hi ha connectats a partir d'una hora determinada i anar veient quins són els seus progressos en el repte.
